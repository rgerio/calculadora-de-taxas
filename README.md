## How to run

Run the following commands in a terminal:

```bash
# Clone this repo
git clone https://github.com/rgerio/calculadora-de-taxas && cd calculadora-de-taxas

# Install dependencies
yarn install
# Mac users/iOS development only
cd ios && pod install && cd ..

# Run the app
yarn android
# or
yarn ios
```
